# Clip Flow

A clipboard history manager with search functionality.

## Installation

1. Clone the repository
2. Install dependencies using uv:
   ```bash
   uv sync
   ```

## Usage

Run the application:
```bash
uv run python main.py
```

## Development

### Linting and Formatting

Format code with Ruff:
```bash
uvx ruff format .
```

Check code style:
```bash
uvx ruff check .
```
